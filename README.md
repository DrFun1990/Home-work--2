# Home-work--2

Данный проект выполнен в рамках домашнего задания для изучения работы с системой контроля версий Git и удалённым репозиторием GitHub.

## Описание проекта
Проект представляет собой простой многостраничный веб-сайт, созданный с использованием HTML. Сайт состоит из главной страницы и нескольких дополнительных страниц, размещённых в отдельной папке.

## Используемые технологии
- HTML5
- Git
- GitHub
- GitHub Pages

## Структура проекта
- index.html — главная страница сайта
- pages/ — дополнительные страницы сайта
- images/ — изображения, используемые в проекте
